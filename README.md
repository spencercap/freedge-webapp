# NYU FREEdge (vue + express)


**installs**

```cli
cd ./
npm install`

```cli
cd ./frontend
npm install
```


**dev frontend**

```cli
cd ./frontend
npm run dev
```

```cli
cd ./frontend
npm run dev
```


**build vue frontend** (builds to ./public folder)

```cli
cd ./frontend
npm run build
```

```cli
cd ./frontend
npm run build`
```


**run server**

```cli
npm start
```


**quick build vue + run server**

```cli
npm run dev
```
