# NYU FREEdge (vue + express)


**installs**

```cli
cd ./
npm install
```

```cli
cd ./frontend
npm install
```


**dev frontend**

```cli
cd ./frontend
npm run dev
```


**build vue frontend** (builds to ./public folder)

```cli
cd ./frontend
npm run build
```


**run server**

```cli
npm start
```


**quick build vue + run server**

```cli
npm run dev
```

**credentials (posting to facebook)**

create local creds.json file for dev or set config vars on server

EX.
```json
{
  "FB_AUTH_TOKEN": "*******",
  "FB_FREEDGE_GROUP_ID": "*******",
  "FB_ALBUM_ID": "*******"
}
```
